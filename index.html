Astro Images - Help

Getting Started

Use File → Open Folder (Ctrl+O) to select a directory containing your astronomical images. The file list will automatically display all supported image formats including FITS, JPEG, PNG, and others.

FITS File Support

Astro Images specializes in FITS (Flexible Image Transport System) files, the standard format for astronomical images:

  • Automatic Processing: FITS files are automatically converted to displayable format
  • Header Parsing: Access to all FITS header metadata via Options → FITS Headers
  • Coordinate Correction: Automatic correction for common FITS data layout issues
  • Histogram Stretching: Enable Auto Stretch for enhanced contrast in faint astronomical objects

Navigation and Viewing

  • Image Selection: Click any file in the list to view it
  • Playback Controls: Use the transport controls for slideshow navigation
  • Zoom Controls: Zoom in/out, fit to window, or view at actual size
  • Auto Stretch: Toggle histogram stretching for better visibility of faint details

File Management

  • Multiple Selection: Ctrl+click or Shift+click to select multiple files
  • Move Files: Use File → Move Selected (Ctrl+M) or the Move button
  • Directory Monitoring: Automatic refresh when files are added/removed

Customization

  • Filename Keywords: Configure custom keywords for filename parsing (Options → Custom Keywords)
  • FITS Headers: Select which FITS header fields to display as columns (Options → FITS Headers)
  • Interface: Resizable sidebar and responsive layout

Keyboard Shortcuts

  • Ctrl+O: Open folder
  • Ctrl+M: Move selected files
  • Ctrl+Plus: Zoom in
  • Ctrl+Minus: Zoom out
  • Ctrl+0: Actual size
  • Ctrl+F: Fit to window
  • F12: Developer tools
  • Ctrl+R: Reload application

Technical Information

Astro Images is built with Electron and uses specialized algorithms for astronomical image processing. The application automatically handles coordinate correction for FITS files and provides advanced caching for improved performance with large astronomical datasets.
